Blockchain defined

First of all it’s important to note that blockchain itself isn’t a device, a machine or a cryptocurrency even. It is a system, a method to keep track of information. The recording of information takes place in a way that makes it impossible to change, hack or cheat the system. Therefore it works well for financial systems like Bitcoin. However it can be used for many more subjects.

Let's build a chain!

As said a blockchain is a system to keep track of blocks of information. Now let’s take the principle of packing your backpack when you are going on holiday. You take one item per time and put it into your suitcase in a certain order.

Every time you add put an item in your suitcase a “transaction” is made. An algorithm calculates a specific number, a hash, based on the item you added to the chain. This number is later used to validate the chain.

As you can see now that chain is not valid anymore. This is because the number calculated for item 3 when we first put information in the chain changed. This is called the proof of work priciple. Because of this principle it is really difficult if not impossible to “hack” a blockchain.
